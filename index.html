About

N.O.
C# 業務アプリ開発｜ASP.NET Web Forms・ASP.NET Coreへ領域を拡張

C#による業務システム開発を主軸とするソフトウェアエンジニアです。 WPF／Windows Formsを用いたWindows業務アプリ開発を中心に、 要件整理、設計、実装、テスト、運用・保守まで一貫して担当してきました。

製造・品質保証・生産管理領域のシステムに携わり、 PLC・ロボット・専用計測器との通信、データ収集、ログ・例外処理、 再送処理など、実運用を考慮した機能の設計・実装に対応してきました。 障害発生時には、ログや通信状況をもとにした原因切り分けと改修も行っています。

直近では、ASP.NET Web Formsによる既存Webシステムの改修に携わり、 画面・設計書の調査、HTML／CSS修正、テスト、リリース対応などを経験しています。

また自己研鑽として、ASP.NET CoreとTypeScript／Next.jsによる 業務想定のWebアプリを開発しています。 認証・認可、API・DB設計、予約・決済、状態遷移、検索・集計、 テスト、CI/CD、クラウドデプロイまで一通り実装しています。

本ポートフォリオでは、実務で培った業務理解、責務分離、 状態管理、例外設計、運用を考慮した実装を、 Webアプリおよびデスクトップアプリの成果物として再現しています。

  • 得意領域： C#業務システム開発（要件整理～運用・保守）
  • 実務領域： WPF／Windows Forms／ASP.NET Web Forms
  • 強み： 外部機器・外部システム連携、ログ・例外・障害切り分けを含む運用設計
  • 自己研鑽： ASP.NET Core／Next.jsによる業務想定Webアプリ開発
