C# 業務アプリ（WPF/WinForms）｜ASP.NET Core/Spring は自己研鑽

業務アプリ開発を主軸とするソフトウェアエンジニアです。 C#（WPF/WinForms）によるWindows業務アプリ開発を中心に、要件整理〜設計〜実装〜テスト〜運用・保守まで一貫して担当してきました。

直近は、PLC連携・TCP通信を含む社内向け業務アプリにおいて、ログ／例外処理／再送など運用を意識した実装と、障害時の原因切り分け・改修にも対応してきました。

併せて自己研鑽として、Java（Spring Boot）やC#（ASP.NET Core）による業務想定の個人開発に取り組み、設計〜実装〜テスト（CI）まで一通り経験しています。

本ポートフォリオでは、実務で培った設計力（責務分離・状態管理・例外設計）を、業務想定の成果物で再現することを重視しています。

  • 得意領域：C# 業務アプリ（要件整理〜運用まで）
  • 強み：外部機器/外部システム連携を前提にした堅牢な実装（ログ・例外・運用設計）
  • 自己研鑽：Spring Boot / ASP.NET Core でWeb領域を拡張
