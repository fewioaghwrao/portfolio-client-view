請求書・入金管理 WPF クライアント（業務ユーザー向け想定）

請求・入金・会員・売上を一元管理

ASP.NET Core Web API と連携する C# / WPF 製のデスクトップクライアントです。 管理者・会員のロールに応じて、請求書確認、入金管理、会員管理、売上確認を行える業務向けアプリとして実装しています。

  • 認証：JWT ログイン、Admin / Member ロール別画面遷移
  • 請求管理：請求書一覧・詳細、入金状況・残額確認
  • 管理機能：会員管理、売上一覧、顧客別集計、催促管理
  • 出力：請求書 PDF 表示、CSV 出力、API連携

About

C#によるWindows業務アプリ開発を主軸に、WPF／WinFormsを用いた業務システムの 要件整理、設計、実装、テスト、運用・保守を担当してきました。 製造・品質保証・生産管理領域での経験があり、画面状態管理、データ処理、 ログ・例外設計、障害切り分けを含む実運用対応を強みとしています。

過去案件では、PLC・ロボット・専用計測器との連携やTCP/IP通信も扱いました。 これらは現在の主軸スキルではなく、製造業システムにおける周辺経験として位置付けています。 現在はASP.NET Web Formsの実務経験を加え、ASP.NET Core／Next.jsによる業務Webアプリ開発も継続しています。

本ページは業務説明会向けの参考資料として作成しています。ご連絡は担当営業（HAL）経由でお願いいたします。
