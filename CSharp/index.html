仮想センサー監視シミュレーション（設備ユーザー向け想定）

リアルタイム可視化 & アラート通知

仮想センサーのストリームを生成し折れ線グラフで可視化。しきい値超過でトースト通知・ログ出力、設定は JSON ホットリロードに対応。

  • 仮想センサー：ノイズ付きサイン波 / ランダムウォーク
  • チャート：OxyPlot、履歴エクスポート（CSV）
  • アラート：しきい値 / デッドバンド、Serilog ログ
  • 設定：appsettings.json ホットリロード、プロファイル切替

Excel / 帳票ランチャー（業務ユーザー向け想定）

テンプレ × データで一括帳票出力

JSONデータを読み込み、Excelテンプレ + ClosedXML / QuestPDF でレポートを一括生成。ジョブはランチャーUIから選択し、設定は JSON ホットリロード対応。

  • テンプレ：Excel（xlsx）と PDF（QuestPDF）の両対応
  • 入出力：JSON 読み込み、出力先ディレクトリ自動採番
  • ログ：実行履歴・エラー出力、失敗ジョブのみ再実行
