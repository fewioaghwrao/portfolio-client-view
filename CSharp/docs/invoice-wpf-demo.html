請求書・入金管理 WPF クライアント — 動画デモ

ASP.NET Core Web API と連携する C# / WPF 製デスクトップクライアントの動作デモです。 管理者向けの請求書確認、入金状況確認、PDF出力、売上確認の流れを中心に紹介します。

※ 動画では、デモ用データを用いて管理者画面の主要な業務フローを確認できます。

デモで確認できる内容

管理者ログイン後、請求書一覧から詳細画面を開き、請求金額・入金済み金額・残額・入金履歴を確認します。 その後、PDF出力や売上一覧など、業務管理に必要な画面を確認する流れです。

デモの流れ

  • 管理者アカウントでログイン
  • 管理者ダッシュボードを表示
  • 請求書一覧を検索・確認
  • 請求書詳細で入金状況と残額を確認
  • PDF出力例を確認
  • 売上一覧・顧客別集計を確認

技術要素

C# .NET 8 WPF MVVM HttpClient ASP.NET Core API JWT OxyPlot xUnit GitHub Actions

補足

  • 社内端末などで実行ファイルのダウンロードや実行が難しい場合に備え、動画で動作確認できるようにしています。
  • デモデータを使用しており、実在する個人情報・顧客情報は含みません。
  • WPF クライアントは既存の ASP.NET Core Web API と連携する別クライアントとして実装しています。
