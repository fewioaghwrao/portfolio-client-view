C# / .NET 8 / WPF / ASP.NET Core API

請求書・入金管理 WPF クライアント — 動画デモ

ASP.NET Core Web APIと連携するC# / WPF製デスクトップクライアントの動作デモです。 管理者向けの請求書確認、入金状況確認、PDF出力、売上一覧・顧客別集計の流れを中心に紹介します。

動画を見る ↓ WPF詳細へ戻る GitHub ↗

※動画では、デモ用データを用いて管理者画面の主要な業務フローを確認できます。 実在する個人情報・顧客情報は含みません。

デモで確認できる内容

管理者ログイン後、ダッシュボードで請求・入金状況を確認し、 請求書一覧から詳細画面を開いて請求金額・入金済み金額・残額・入金履歴を確認します。 その後、PDF出力、売上一覧、顧客別集計などの業務画面へ進みます。

デモの流れ

  • 管理者アカウントでログイン
  • 管理者ダッシュボードを表示
  • 請求書一覧を検索・確認
  • 請求書詳細で入金状況と残額を確認
  • 請求書PDFを表示
  • 売上一覧・顧客別集計を確認

技術要素

C# .NET 8 WPF MVVM HttpClient ASP.NET Core API JWT OxyPlot xUnit GitHub Actions

補足

  • 実行ファイルを起動しにくい環境でも、動画から動作を確認できます。
  • 掲載している請求・会員・入金情報はすべてデモ用です。
  • WPF版は既存のASP.NET Core Web APIを利用する別クライアントとして実装しています。
